# 提灯 (Lantern) 项目理解

> 远游山川，星河在天。

## 项目定位

**本地优先、AI 驱动的人生管理系统**（Tauri 2 桌面/移动应用）

核心价值：
- 不是"被催促完成任务"，而是"被陪伴着走过夜路"
- 把任务、日记、日历、习惯、人脉、成长融为一体
- RPG 式六维属性系统，让每一点努力都被看见

## 世界观隐喻

| 意象 | 含义 |
|------|------|
| 夜路 | 人生有暗处 |
| 提灯 | AI 陪伴、日记、记忆——散落的光 |
| 萤火 | 故人的化身，地上的星星，每段记忆 |
| 桑启 | 用户本人，在夜路里行走，收集自己的故事 |
| 祖树 | 记忆的载体，故事不会消失 |
| 星空 | 远方的希望 |

## 设计关键词

**静谧、陪伴、诗意**

- 深色背景 `#0F1412` = 夜空，不是纯黑，有呼吸感
- 萤火绿 `#4CAF76` = 主色，不刺眼
- 毛玻璃 + 透明度分层 = 月光薄雾
- 动效"轻、慢、柔" = 像萤火虫的飞行，不是闪电

## 技术架构

```
Tauri 2 (Rust + React/TypeScript)
├── src-tauri/
│   ├── ai/           # AI 子系统（47 个工具）
│   ├── commands/     # IPC 命令层
│   ├── db/           # SQLite (WAL 模式)
│   └── sync/         # WebDAV 同步（坚果云）
└── src/
    ├── components/   # UI 组件
    ├── pages/        # 页面
    ├── stores/       # Zustand
    └── styles/       # 主题配置
```

**数据存储原则**：
- 日记用 `.md` 文件（文件系统为真相源）
- 其他数据用 SQLite，单文件 `lantern.db`
- 用户可随时用外部工具读取所有数据
