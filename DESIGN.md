# 提灯设计规范

> 远游山川，星河在天。

---

## 一、设计哲学

### 1.1 提灯的故事

鸣沙之谷有一棵祖树，树上住着无数萤火。

族人相信，萤火是故人的化身，也是地上的星星。孩子们围坐在祖树下，听长辈讲述外面的故事，萤火在人群中飞舞，有若漫天星河洒落。

后来，一个叫桑启的孩子带上故事书和木剑，穿过石林迷阵，去外面的世界收集新的故事。等他回来时，家乡已在岁月中消逝，祖树也已枯萎。

他没有陷在悲伤里。他学着父亲的样子，在枯萎的祖树下，缓缓讲述他收集的新故事。

点点萤火自枯萎的树干中出现，环绕着他飞舞。

父亲说："最重要的东西，只有用心才能看见。"
父亲还说："不要害怕迷失方向，星星会照亮我们的归途，而萤火，就是地上的星星。"

桑启终于懂了——故事不会消失。记忆化作萤火，萤火化作星光。种子洒在途中，新的树苗正在生长。

"没关系，我知道，鸣沙还在这里。"

### 1.2 提灯是什么

提灯是这个故事的延续。

- 你就是桑启，在夜路里行走，收集自己的故事
- 提灯是你的萤火——AI 陪伴、日记、记忆，都是散落的光
- 你走过的地方，都亮了

### 1.3 提灯不是什么

- 不是效率至上的 GTD 工具——不催你、不评判你
- 不是社交平台——故事是讲给自己听的
- 不是冰冷的数据看板——每个数字背后都是一个人、一件事、一段路
- 不用"完成率""待办数"制造焦虑——桑启从不焦虑，他只是走着、记着

### 1.4 设计原则

**夜的隐喻**：默认深色背景 = 人生有暗处，但你手里有光。不需要驱散所有黑暗，照亮脚下这一步就够了。

**低压力自然主义**：拒绝高饱和红色、刺眼提醒、紧迫感。用低饱和、柔和的色调传递平静。用户打开提灯不是"要完成任务"，而是"看看我的小森林"。

**呼吸感**：界面像夜里的微风，不是瀑布。留白是给眼睛休息的地方，不是浪费的空间。

**故事思维**：所有功能的底层逻辑是"记录故事"，不是"管理数据"。任务是一颗种子，日记是一段旅途，记忆是飞舞的萤火。

### 1.5 语气调性

| 场景 | 语气 | 示例 |
|------|------|------|
| AI 对话 | 散落的短诗，轻盈有余韵 | "今天的日记，像是一封没写完的信。" |
| 空状态 | 温暖邀请，不催促 | "夜还长，不急。先坐下来，说说今天？" |
| 完成反馈 | 平静满足，不夸张 | 微光闪烁一下，不弹窗不撒花 |
| 错误提示 | 温和，给台阶 | "这条路好像走不通，要换一条试试吗？" |
| 删除确认 | 尊重，不轻率 | "这个故事一旦删掉，就不能回来了。确定吗？" |
| 欢迎语 | 像老友重逢 | "你来了。今天想记点什么？" |

## 二、配色系统

### 2.1 色板：「夜萤」

所有颜色围绕"夜空、微光、自然"设计。低饱和、柔和、不压迫。

鸣沙的人相信，萤火是地上的星星。色板的灵感来自：夜空的深沉、萤火的微光、祖树的绿意、沙土的温暖。

#### 暗色主题（默认）

| Token | 色值 | 用途 | 设计理由 |
|-------|------|------|---------|
| `--bg-deep` | `#0F1412` | 页面主背景 | 墨绿夜空，不是纯黑，有呼吸感，像鸣沙之谷的夜 |
| `--bg-card` | `rgba(255,255,255,0.04)` | 卡片背景 | 极低透明度白，毛玻璃质感，如月光薄雾 |
| `--bg-elevated` | `rgba(255,255,255,0.07)` | 弹窗、浮层 | 稍高的透明度，区分层级 |
| `--accent` | `#4CAF76` | 主强调色 | 萤火绿，祖树的光晕，不刺眼 |
| `--accent-glow` | `#4CAF76` at 15% | 光晕/辉光 | 提灯周围的环境光 |
| `--text-primary` | `rgba(255,255,255,0.88)` | 主文字 | 不用纯白，降低对比度保护眼睛 |
| `--text-secondary` | `rgba(255,255,255,0.50)` | 次要文字 | 降低层级，不干扰主信息 |
| `--text-tertiary` | `rgba(255,255,255,0.30)` | 辅助文字 | 时间戳、状态标签 |
| `--border` | `rgba(255,255,255,0.06)` | 分割线 | 极淡，仅暗示边界 |
| `--firefly` | `#A8E6CF` | 萤火虫光点 | 比主色更亮更暖，故事的光 |
| `--firefly-dim` | `#6B9DC2` | 萤火虫暗色 | 萤火虫呼吸动画的低点，星光色 |
| `--warm` | `#D4A76A` | 温暖色 | 篝火、沙土、故事书的纸页 |

#### 亮色主题（清晨）

| Token | 色值 | 用途 |
|-------|------|------|
| `--bg-deep` | `#F4F7F5` | 页面主背景，清晨薄雾绿 |
| `--bg-card` | `#FFFFFF` | 卡片背景 |
| `--accent` | `#2D6B4F` | 主强调色，深森绿 |
| `--text-primary` | `#1A2E23` | 主文字 |
| `--text-secondary` | `#5C7A6A` | 次要文字 |

### 2.2 功能色

| 用途 | 暗色 | 亮色 | 说明 |
|------|------|------|------|
| 成功/完成 | `#4CAF76` | `#2D6B4F` | 与主色一致，萤火闪一下 |
| 警告 | `#D4A76A` | `#B8860B` | 柔陶橙，篝火的颜色 |
| 危险/删除 | `#C97070` | `#A04040` | 低饱和红，不用高亮红 |
| 信息 | `#6B9DC2` | `#4A8DB8` | 浅露蓝，星光色 |

## 三、排版系统

### 3.1 字体

```
--font-display: SF Pro Display, system-ui, -apple-system, BlinkMacSystemFont, sans-serif
--font-body: SF Pro Text, system-ui, -apple-system, BlinkMacSystemFont, "Microsoft YaHei", sans-serif
```

无衬线，干净现代，像桑启的故事书——不华丽，但每个字都看得清。

### 3.2 字号层级

| 层级 | 字号 | 字重 | 行高 | 字间距 | 用途 |
|------|------|------|------|--------|------|
| 页面标题 | 28px | 600 | 1.2 | +0.3px | 模块大标题 |
| 区块标题 | 20px | 500 | 1.3 | 0 | 子模块标题 |
| 卡片标题 | 16px | 500 | 1.4 | 0 | 任务名、日记标题 |
| 正文 | 14px | 400 | 1.6 | 0 | 描述、内容 |
| 辅助文字 | 12px | 400 | 1.4 | 0 | 时间、状态标签 |
| 微型标签 | 10px | 500 | 1.2 | +0.5px | 角标、计数 |

### 3.3 排版原则

- 大标题字间距宽松（+0.3~1px），传递开阔感，像站在石林外第一次看到云中漠地
- 正文行高 1.6，阅读不费力，像听桑启讲故事的节奏
- 深色背景上不用纯白文字，用 `rgba(255,255,255,0.88)` 降低对比度

## 四、组件系统

### 4.1 圆角

| 元素 | 圆角 | 说明 |
|------|------|------|
| 卡片 | 20px | 模拟祖树叶片的柔和曲线 |
| 按钮 | 9999px（pill） | 胶囊形，无攻击性 |
| 输入框 | 14px | 柔和但有边界 |
| 图标容器 | 14px | 统一 |
| 弹窗 | 24px | 大圆角，包容感 |
| 模态框 | 20px | 与卡片一致 |

### 4.2 阴影与分层

**暗色主题下不用传统阴影**，用透明度分层——像夜里的雾气，自然地分出远近：

| 层级 | 背景 | 用途 |
|------|------|------|
| L0 底层 | `--bg-deep` | 页面背景，夜空 |
| L1 卡片 | `rgba(255,255,255,0.04)` | 卡片、列表项 |
| L2 浮层 | `rgba(255,255,255,0.07)` | 弹窗、下拉菜单 |
| L3 顶层 | `rgba(255,255,255,0.10)` | 模态框、确认对话框 |

**亮色主题**用极浅阴影：`0 2px 12px rgba(0,0,0,0.05)`

### 4.3 毛玻璃

```css
.glass {
  background: rgba(255, 255, 255, 0.04);
  backdrop-filter: blur(20px) saturate(1.2);
  -webkit-backdrop-filter: blur(20px) saturate(1.2);
  border: 1px solid rgba(255, 255, 255, 0.06);
}
```

用于：导航栏、浮层、卡片叠加层。像隔着一层薄雾看夜空。

### 4.4 间距

8px 基础网格：

| 用途 | 间距 |
|------|------|
| 元素内边距 | 16px |
| 卡片内边距 | 20px |
| 区块间距 | 24px |
| 页面边距 | 24px（移动端 16px） |
| 列表项间距 | 8px |

## 五、提灯视觉语言

### 5.1 提灯图标

- SVG 组件：`LanternSvg`
- 灯体轮廓：`rgba(255,255,255,0.12)` 描边，无填充
- 灯光晕：`--accent` 径向渐变，低透明度
- 眼睛：`--accent` 绿色竖线，追踪鼠标 + 眨眼

提灯就是桑启的木剑——看起来普通，但藏着故事的光。

### 5.2 萤火虫

> 鸣沙的人相信，萤火是故人的化身，也是地上的星星。

萤火虫是提灯的视觉延伸。每一个萤火虫代表一段记忆、一篇日记、一个故事。

**视觉规则：**
- 3~7 个光点，尺寸 2~5px
- 颜色：`--firefly` (#A8E6CF) 到 `--firefly-dim` (#6B9DC2) 呼吸渐变
- 运动：缓慢漂浮，轨迹不规则，模拟真实萤火虫
- 出现场景：提灯页面空状态、加载状态、日记完成时

**动画参数：**
```css
@keyframes fireflyFloat {
  0%, 100% { transform: translate(0, 0); opacity: 0.3; }
  25% { transform: translate(10px, -15px); opacity: 0.8; }
  50% { transform: translate(-5px, -25px); opacity: 0.5; }
  75% { transform: translate(15px, -10px); opacity: 0.9; }
}

@keyframes fireflyGlow {
  0%, 100% { box-shadow: 0 0 4px 2px rgba(168, 230, 207, 0.3); }
  50% { box-shadow: 0 0 8px 4px rgba(168, 230, 207, 0.6); }
}
```

- 每个萤火虫动画时长 6~12s，随机延迟
- 漂浮范围：以提灯为中心，半径 100~200px

### 5.3 光晕

提灯和萤火虫周围有柔和的环境光，像桑启在祖树下被萤火环绕的样子：

```css
.lantern-aura {
  background: radial-gradient(
    ellipse at center,
    rgba(76, 175, 118, 0.15) 0%,
    rgba(76, 175, 118, 0.05) 40%,
    transparent 70%
  );
}
```

### 5.4 星空

在提灯页面的深色背景中，可以有极微弱的星点装饰（1~2px 白点，opacity 0.1~0.3），散落在远处。代表桑启说的"星河在天"——远方的希望。

不抢萤火虫的视觉焦点，只是安静地在背景里。

## 六、动效原则

### 6.1 核心规则

动效要"轻、慢、柔"。像萤火虫的飞行，不是闪电。像桑启讲故事的语调——轻缓柔和。

| 动效 | 参数 | 意图 |
|------|------|------|
| 页面切换 | fade + 轻微滑动，300ms ease-out | 像风吹过树叶 |
| 卡片点击 | scale(0.98)，150ms | 轻微反馈，不突兀 |
| 任务完成 | 光晕闪一下，400ms | 平静满足，不撒花 |
| 弹窗出现 | fade-in + scale(0.95→1)，250ms | 柔和浮现 |
| 萤火虫 | 6~12s 漂浮循环 | 持续的微动，夜的氛围 |

### 6.2 呼吸感

关键元素有轻微的"呼吸"动画（透明度/尺寸缓慢脉动）：
- 提灯的光晕：`glowPulse` 2s 循环
- 萤火虫：独立的漂浮 + 发光循环
- 选中状态的指示器：缓慢脉动

## 七、暗色主题实现规范

### 7.1 背景层级

```
页面背景    #0F1412           ← 夜空
卡片背景    rgba(255,255,255,0.04) + backdrop-blur  ← 月光薄雾
浮层背景    rgba(255,255,255,0.07)
模态背景    rgba(255,255,255,0.10)
遮罩层      rgba(0,0,0,0.5)
```

### 7.2 文字对比度

- 主文字：`rgba(255,255,255,0.88)` — 在 `#0F1412` 上对比度约 12:1
- 次要文字：`rgba(255,255,255,0.50)` — 对比度约 6.5:1
- 辅助文字：`rgba(255,255,255,0.30)` — 对比度约 3.8:1（仅用于装饰性文字）

### 7.3 边框与分割

- 不用实色边框，用 `rgba(255,255,255,0.06)` 的 1px 线
- 分割用透明度差异，不用线条（毛玻璃卡片叠在深背景上自然分层）

### 7.4 输入框

```css
.input-dark {
  background: rgba(255, 255, 255, 0.06);
  border: 1px solid rgba(255, 255, 255, 0.08);
  border-radius: 14px;
  color: rgba(255, 255, 255, 0.88);
}
.input-dark:focus {
  border-color: var(--accent);
  box-shadow: 0 0 0 3px rgba(76, 175, 118, 0.15);
}
```

## 十、品牌语

- **中文名**：提灯
- **英文名**：Lantern
- **tagline**：远游山川，星河在天。
- **世界观**：人生是一段夜路。你提着灯，灯不大。周围有萤火，远处有星空。你走过的地方，都亮了。
- **承诺**：故事不会消失。你记下的每一个字，都是一颗种子。种子洒在途中，终会长成大树。
